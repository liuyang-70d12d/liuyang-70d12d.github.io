Paper Note: Learn to Grow: A Continual Structure Learning Framework for Overcoming Catastrophic Forgetting

Learn to Grow: A Continual Structure Learning Framework for
Overcoming Catastrophic Forgetting

Xilai Li, Yingbo Zhou, Tianfu Wu, Richard Socher, Caiming Xiong

ICML 2019

[paper]

Main Idea

此前使用DNN的持续学习方法通常对于所有任务都是使用相同的模型结构，本文作者提出要明确（explicitly）地学习 task specific 的模型结构。本文其实就是在用NAS来做持续学习，为每个任务寻找合适的网络结构。

Figure 1. (a) 以 EWC (Kirkpatrick et al., 2017).为代表的一类方法，除了 task specific 层以外其它参数都共享，通过防止重要参数改变过大来避免灾难性遗忘。(b) 每个任务都会给网络添加一些 task specific 的参数，所有层之前的参数是被固定的，以此来防止遗忘，PackNet 好像是这么个意思。© 本文提出的方法，即通过结构搜索来学习得到网络结构。以该图为例，搜索结果决定复用前两层，对第3层做 “adaptation”，并分配新的权重给第4层。
