Reinforced Continual Learning

Ju Xu, Zhanxing Zhu

NeurlIPS 2018

[paper] [code]

Main Idea

文章提出了一种新的基于强化学习的持续学习方法。这种持续学习方法的思路也是通过动态扩增网络结构来学习新的任务。当遇到新的任务时，决定为神经网络的每一层增加 nodes 或者 filters 的最佳数目可以看作是一个组合优化问题。我们用强化学习来解决这一问题，因此将其称为 Reinforced Continual Learning (RCL) 。

在 RCL 中，使用一个由RNN网络实现的 controller 来决定神经网络相应于每个任务的最佳的结构超参数（architectural hyper-parameters）。我们使用 actor-critic 策略来训练这个 controller，由验证准确率和网络复杂度来提供 reward 信号。这样在保持模型复杂度较小的同时，也尽可能保证了模型在旧任务上的预测准确率。

Method

RCL 包括了三个网络：controller，value network 以及 task network。controller 由LSTM实现，用来生成 policies 以及决定为每个任务增加多少 filter 和 nodes。value network 由全连接网络实现，用于估计状态的值。task network 可以是任意网络，用于解决某个特定的任务（在本文中 task network 是由 CNN 实现的）。
