Gradient Projection Memory for Continual Learning

Anonymous

ICLR 2021 Conference Blind Submission

[paper]

Background

文章首先反思了神经网络为什么会出现“灾难性遗忘”，文章认为神经网络本质上就是把训练数据分布的信息编码到网络的参数化的表示当中。当遇到新的任务时，基于梯度的优化方法，由于没有任何限制，就会根据当前的数据分布改变了已经习得的编码，以此来最小化目标函数。这样的参数更新方式导致了“遗忘”。

Related Work

这篇工作是顺着GEM、A-GEM这条线做的，这条线上的工作还包括：GEM (Lopez-Paz & Ranzato, 2017)、Zeng et al. (2018)、A-GEM (Chaudhry et al., 2019a)、Chaudhry et al. (2019b)、Guo et al. (2020)、Farajtabar et al. (2020)。文章认为之前的工作要么保留原始数据，存在隐私泄露的风险，要么资源利用率不高，可扩展性不强，所以提出了这篇工作。

文章将持续学习的方法分为了三个类别：

Expansion-based methods

Regularization-based methods

Memory-based methods

与这篇文章最相关的工作包括 Orthogonal Gradient Descent (OGD) (Farajtabar et al., 2020) 、Orthogonal Weight Modulation (OWM) (Zeng et al., 2018)。

Main Idea

正如在 Related Work 中所提到的，本文提出的解决“灾难性遗忘”问题的方法也是对梯度方向施加了显性的限制，但不同于此前的方法，本方法既不需要存储旧的数据也不需要存储旧的梯度方向。该方法是，在学习完每一个任务之后，将整个权重的梯度空间划分为两个子空间：Core Gradient Space (CGS) 和 Residual Gradient Space (RGS)。

利用输入和梯度空间之间的关系，我们展示了习得的“表征”如何构成了全连接网络和卷积网络中这些梯度子空间的基（bases）。在这些“表征”上使用奇异值分解（SVD），我们展示了如何获取CGS的“基”的最小集，通过该集可以保留过去的知识并确保新任务的可学习性。我们将这些"基"存储在 Gradient Projection Memory (GPM) 中。我们通过在由 GPM 张成的CGS 正交的方向上更新梯度来学习新任务。我们的分析表明，这样的正交梯度下降对旧任务的影响很小，甚至没有干扰，因此可以有效地减轻灾难性遗忘。我们证明了我们的方法具有存储高效性，并且可以扩展到具有更长任务序列的复杂数据集，同时又可以保护数据隐私。

Method

Input and Gradient Spaces

我们的算法基于以下事实：stochastic gradient descent (SGD) updates lie in the span of input data points (Zhang et al., 2017)。

Continual Learning with Gradient Projection Memory (GPM)

本文提出的持续学习算法就是利用了上面所阐述的梯度与输入空间之间的关系，来识别过往任务的核心梯度空间（core gradient spaces）。

Learning Task 1

我们在学习第一个任务的时候，对参数更新没有任何限制。在 task 1结束的时候，为了保留所学任务的知识，我们添加了对后续任务梯度更新方向的限制。为此，我们将整个梯度空间划分为两个正交的子空间：Core Gradient Space (CGS) 和 Residual Gradient Space (RGS)，也就是说在CGS上的梯度变化会导致对所学任务有很大的干扰，而在 RGS 上的梯度变化则不会。我们的目的是找到并存储 CGS 的基（base），并在学习下一个任务的时候，梯度更新的方向与CGS正交。在我们的构想中，每一层都有它自己的 CGS。

……

文章后面的内容就不仔细说了，意思还是比较简单的。为了找到基，在学习完 task 1后，我们为每一层构建一个表征矩阵（representation matrix），这个表征矩阵包含了梯度与输入空间的关系。使用SVD来得到 CGS 的基，为了避开CGS，就是把后面任务的梯度更新减去CGS的基的分量，以实现正交的目的，也就是RGS。CGS的基在文中被称为 Gradient Projection Memory (GPM)，模型需要存储的是且仅是GPM。随着不断学习新的任务，GPM也会随之增加。
