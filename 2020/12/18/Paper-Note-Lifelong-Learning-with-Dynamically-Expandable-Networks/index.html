Lifelong Learning with Dynamically Expandable Networks

Jaehong Yoon, Eunho Yang, Jeongtae Lee, Sung Ju Hwang

ICLR 2018

[paper] [code] [video]

Main Idea

本文提出的方法是在遇到每个任务 ttt 都会重新训练网络，使得每个新任务都能够利用并只改变之前所训练网络的与之相关的那一部分，并且允许在必要的时候继续扩展网络容量。换言之，每个任务 ttt 都会使用与此前任务不同的子网络，但仍会与此前任务共享相当一部分子网络。Figure 1将本篇文章的工作与之前的持续学习方法进行了比较。

Figure 1. 神经元和权重如果是红色的，表示被重新训练过，如果是黑色，表示被固定住了，没有被重新训练。在遇到新的任务 t 时：(a) 模型只被重新训练过而没有扩展结构，比如 Elastic Weight Consoliation (Kirkpatrick et al. 2017)；(b) 模型只是扩展了结构而没有重新训练，比如 Progressive Network (Rusu et al., 2016)；© 模型既会被重新训练，在必要时也会扩展结构，比如本篇工作 DEN。

这个方法的提出需要解决以下三个问题：(1) 随着网络结构的扩展，训练的计算开销也会随之增加，如何将计算开销压低在一个较低水平；(2) 前文讲到是“在必要时”扩展网络，那么如何判断什么时候是“必要时”，需要增加多少神经元；(3) 如何避免语义漂移（semantic drift）和灾难性遗忘，既要重新训练网络，又要添加新的神经元，如何避免它们对已有网络模型的影响。

本文为了解决以上问题，提出了 Dynamically Expandable Networks (DEN)。

Method

为求行文简洁，文章只关注了二分类任务，但是文章所提出的方法是适用于任何任务的。持续学习在 ttt 时刻通过优化下面的式子来学习模型参数 Wt\boldsymbol{W}^tWt ：

min⁡WtL(Wt;Wt−1,Dt)+λΩ(Wt),t=1,...(1)\min\limits_{\boldsymbol{W}^t} \mathcal{L}(\boldsymbol{W}^t;\boldsymbol{W}^{t-1},\mathcal{D}_t) +\lambda\Omega(\boldsymbol{W}^t),\quad t=1,...\qquad\qquad(1) Wtmin​L(Wt;Wt−1,Dt​)+λΩ(Wt),t=1,...(1)

Selective Retraining

Selective Retraining 是希望当模型学习新的任务的时候，网络找到与该任务相关的神经元，只对它们进行重新训练。

Dynamic Network Expansion

Dynamic Network Expansion 是希望当训练时loss高过某一个阈值τ\tauτ，就在网络的每一层都扩展 k 个神经元，再将其中没有用的删掉。

但是我们其实并不想在每一层都固定地添加 kkk 个神经元，这样效率显然不够高。DEN使用组稀疏正则（group sparse regularization）来动态地决定在网络的每一层添加多少神经元：

min⁡WlN L(WlN;Wlt−1,Dt)+μ∣∣WlN∣∣1+γ∑g∣∣Wl,gN∣∣2(5)\min\limits_{\boldsymbol{W}^{\mathcal{N}}_{l}}\ \mathcal{L}(\boldsymbol{W}^{\mathcal{N}}_{l};\boldsymbol{W}^{t-1}_{l},\mathcal{D}_t) +\mu||\boldsymbol{W}^{\mathcal{N}}_{l}||_1 +\gamma\sum\limits_g||\boldsymbol{W}^{\mathcal{N}}_{l,g}||_2 \qquad\qquad(5) WlN​min​ L(WlN​;Wlt−1​,Dt​)+μ∣∣WlN​∣∣1​+γg∑​∣∣Wl,gN​∣∣2​(5)

Network Split/Duplication

这一部分是为了解决语义漂移（semantic drift）和灾难性遗忘问题的。

文章称这种方法为 split ，即“切分”神经元，使得可以对两个不同的任务都有最优的特征表示。具体方法如下：首先按照Eq.6进行网络优化，然后对每个隐藏层的神经元 iii 的语义漂移量 ρit\rho_i^tρit​ 进行度量，也就是去度量 t−1t-1t−1 和 ttt 时的输入权重的 l2l_2l2​-distance。如果 ρit\rho_i^tρit​ 大于阈值 σ\sigmaσ ，我们认为在之前的训练过程中，特征的语义发生了较大的变化。我们就将神经元 iii “分割”为两个副本。完成对神经元的复制之后，再用 Eq.(6) 去训练网络，因为此时网络结构发生了变化，需要再去训练。其中 Eq(6) 如下：

Timestamped Inference
