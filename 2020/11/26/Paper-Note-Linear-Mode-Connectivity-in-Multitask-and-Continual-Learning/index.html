Paper Note: Linear Mode Connectivity in Multitask and Continual Learning

Created2020-11-26Updated2021-01-25Visited

Linear Mode Connectivity in Multitask and Continual Learning

Seyed Iman Mirzadeh, Mehrdad Farajtabar, Dilan Gorur, Razvan Pascanu, Hassan Ghasemzadeh

ICLR 2021

paper

Main Idea

在这项工作中，我们从一个非常规的问题开始，即“在多任务学习和持续学习的最小值之间可能存在局部几何特性方面的什么关系”。我们的工作受到最近关于模式连接（mode connectivity）的研究的启发 (Draxler et al., 2018; Garipov et al., 2018; Frankle et al., 2020) ，这些研究发现，通过基于梯度的优化方法获得的不同最优值是可以通过不增加损失值的简单路径连接的。我们尝试了解多任务学习和持续学习是否也能通过低损失的 manifold 连接在一起，并且该 manifold 可以采用的最简单形式是什么。

我们发现（如 Fig. 1 所示），只要像下面所述的那样，多任务学习与持续学习共享相同的初始化，linear manifold 能够将多任务学习可靠地连接到持续学习。 就通过 loss landscapes 和优化轨迹的角度来理解灾难性遗忘现象以及设计更好的持续学习算法而言，这是一个重大发现。

Figure 1.

我们考虑 Fig. 1 左侧所示的特定学习方式：使用训练数据集 D1\mathcal{D}_1D1​ 学习完第一个任务之后，兵分两路：1）在此基础上继续训练数据集 D2\mathcal{D}_2D2​ ，即持续学习第二个任务，得到模型 $ \hat{w}_2$；或者2）同时训练 D1+D2\mathcal{D}_1+\mathcal{D}_2D1​+D2​，得到多任务模型 w2∗w^*_2w2∗​。我们将研究 w^2\hat{w}_2w^2​ 和 w2∗w_2^*w2∗​ 之间的关系。注意 w2∗w_2^*w2∗​ 不是由典型意义上的多任务学习得到的，典型的持续学习应该从 w0w_0w0​ 开始，然后在两个数据集上训练。

对于为什么要按 Fig.1 这样去设计setting，论文也给出了三点解释：

  1. 我们选择了这种稍微非常规的 setting，以最大程度地减少导致两种解决方案之间差异的混淆因素(Fort et al., 2019)。
  2. 我们还基于 (Frankle et al., 2020) 得到的观察——初始化可能会对在同一任务上的解决方案之间的连通性产生很大影响。共享相同的起点，就像我们对 w^2\hat{w}_2w^2​ 和 w2∗w^*_2w2∗​ 所做的那样，可能会保证这两种解决方案之间的损失值很小的 linear path。
  3. 此外，(Neyshabur et al. 2020) 指出，在迁移学习的背景下，从预训练权重开始的两个最小值之间没有 performance barrier，这表明预训练权重将优化引导到了 loss landscape 的平坦盆地之中。相反，如果这两个最小值从随机初始化的权重开始，则显然存在 barriers。

论文就是在探究 w^2\hat{w}_2w^2​ 与 w2∗w^*_2w2∗​ 之间的关系。更为笼统地讲，是在探究 continual learning 与 multitask learning 之间的关系。

在此基础上，建立了 continual learning 与 multitask learning 之间的一种连接性—— the connectivity through a manifold of low error。由此，论文提出了一种新的 CL 算法：Mode Connectivity SGD (MC-SGD)。
